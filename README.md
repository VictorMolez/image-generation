# image-generation
Pour expérimenter, j'ai créé et entrainé un modèle de diffusion sur **Kaggle** pour générer des images.

J'ai créé les données d'entrainement artificiellement. Il s'agit de 20000 images avec un texte écrit dessus. Je n'ai mis que 50 images en exemple sur GitHub.
Il me reste beaucoup d'améliorations à effectuer, dont la création d'un dataset plus simple.
